## 🎨 À propos

ImprimeNéo est une plateforme d'impression professionnelle haute qualité opérant en Belgique, France et Luxembourg. Nous offrons des solutions d'impression complètes pour vos besoins en marketing et communication.

## 🚀 Fonctionnalités

- ✅ Impression haute qualité (Noir & Blanc, Couleur)
- ✅ Cartes de visite professionnelles
- ✅ Affiches publicitaires
- ✅ Dépliants et brochures
- ✅ Livraison rapide (24-48h)
- ✅ Paiement sécurisé
- ✅ Service client réactif
- ✅ Devis gratuit sans engagement

## 🛠️ Installation

### Prérequis
- Node.js >= 14.0.0
- npm ou yarn

### Étapes

```bash
# Cloner le repository
git clone https://github.com/imprimerieneo/Imprimerien-o.git
cd Imprimerien-o

# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Build pour la production
npm run build
```
